s[n]printf函数fmt漏洞

Sep 10, 2025

函数原型 

1

int sprintf(char *str, const char *format, ...);

str:目标字符数组，用于储存格式化后的结果

format:格式化字符串，包含普通字符和格式符，如a=%d

…:可变参数，对应格式字符串中的格式符，类型需与格式符匹配。如 若参数format为a=%d则…应为int类型的整数

返回值是向str中输入的字符个数

1

int s[n]printf(char *str, size_t size, const char *format, ...)

比sprintf 多了一个size_t参数，用于限制可写入字符数

#include <stdio.h>
#include <string.h>  

int main() {
    char buf[200];  
    int num = 123;
    float f = 3.14159;
    char str[] = "hello";
    int arr[] = {1, 2, 3};  

    // 1. 格式化整数、字符串、浮点数
    int len1 = sprintf(buf, "整数: %d, 字符串: %s, 浮点数: %.2f", num, str, f);
    printf("结果1: %s\n", buf);
    printf("写入字符数: %d (验证: %zu)\n\n", len1, strlen(buf));  // 不含 \0

    // 2. 格式化十六进制、地址
    int len2 = sprintf(buf, "十六进制: 0x%x, 数组地址: %p", num, arr);
    printf("结果2: %s\n", buf);
    printf("写入字符数: %d (验证: %zu)\n\n", len2, strlen(buf));

    // 3. 拼接多个数据
    int len3 = sprintf(buf, "综合: num=%d, str=%s, f=%.1f, arr@%p", num, str, f, arr);
    printf("结果3: %s\n", buf);
    printf("写入字符数: %d (验证: %zu)\n", len3, strlen(buf));

    return 0;
}

漏洞原理

攻击者通过溢出等手段可修改format参数，完整任意地址读写

PS:snprintf是把格式化字符串的地址记下来，每次要处理一个字符时，先从地址处取格式化字符串，然后再根据格式化字符串来处理字符
